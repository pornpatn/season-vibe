
# Season Thai Core App

Includes:
1. Project Setup (Monorepo)
2. Custom Material UI Theme
3. Multi-language Support (EN/TH/ES)
4. User Management with JWT

## Run Instructions

### Backend
```bash
cd apps/backend
npm install
npm start
```

### Frontend
Use any dev server (e.g., `npx serve src`)
```bash
cd apps/frontend
npx serve src
```
